- 聊聊 undefined 和 null 的区别
答案：
相同点 => 都没有属性和方法，
不同点 => 
  undefined 代表未定义，已经申请栈内存，并进行了一定的初始化，但是没有指向堆对象
  null 代表空或者置空，可以理解为，已经申请了栈内存，但没有初始化，也没有指向堆对象
