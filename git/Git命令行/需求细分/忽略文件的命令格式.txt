#  注释符
*  通配符
/  路径符
[] 或选符



例如：
*.[oa]    --匹配所有以o或者a为后缀的文件
/TODO     --匹配根目录下的TODO文件夹
TODO/     --匹配TODO下的所有文件

【例子】

*.[oa]		--忽略*.o和*.a文件


*.[bB]		--忽略*.b和*.B文件，my.b除外
!my.b


dbg		--忽略dbg文件和dbg目录			


dbg/		--只忽略dbg目录，不忽略dbg文件


dbg		--只忽略dbg文件，不忽略dbg目录
!dbg/


/dbg		--只忽略当前目录下的dbg文件和目录，子目录的dbg不在忽略范围内
