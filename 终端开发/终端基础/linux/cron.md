# 定义时间周期

- 格式: `* * * * *` 从左到右，分别代表 【分】 【时】 【日期】【月份】【星期】

- 特殊字符:
1. `*` 代表默认，未定义
2. `/` 代表每个单位时间; `/5` 代表每5个单位时间
3. `-` 代表一个区间
4. `,` 分割符

- 样例说明：
> `0 6 * * *` => 每天早上6点 

> `0 */2 * * *` => 每2个小时

> `0 23-7/2,8 * * *` => 晚上11点到早上8点之间每2个小时和早上8点

> `0 11 4 * 1-3` => 每个月的4号和每个礼拜的礼拜一到礼拜三的早上11点
