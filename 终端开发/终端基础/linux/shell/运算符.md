# 数学运算符
支持 `+ - * %`
```shell
a=1
b=2
c=`expr $a + $b` # 注意必须有空格，否则会当作单个字符串处理
d=`expr $a+$b` # 输出 '1+2'
```

# 关系运算符

关系运算符只支持 数字的比对，不支持字符串

- `-eq` 相等
- `-ne` 不相等
- `-gt` 大于
- `-lt` 小于
- `-ge` 大于等于
- `-le` 小于等于

# 布尔运算符
- `&&`	与
- `||`	或
	
``` bash
a=1
b="1"
if [[ $a -eq $b || $a -lt $b ]] # 注意空格是必须的，否则报错
then 
  echo  'expr exec'
else [ $a -gt $b ]
  echo `$a -gt $b`
```
