1. 只记录超过1000手以上的单字
2. 记录有万手单的票子，第二天行情如何，其中区分净流入和净流出的情况对比
3. 连续万手单和偶尔万手单，对第二天的影响

# 完整数据样本存储模型
``` yaml
`${stock}`: {
	`${date}`: {
		time: '', # 时间
		hands: '', # 交易手数
		price: '', # 交易单价
	}
}
```
